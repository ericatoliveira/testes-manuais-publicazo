UC004 - Reservar um serviço

Descrição

  • Este caso de uso permite que usuários cadastrados reservem algum serviço disponível.

Perfil

  • Cliente.

Pré-condições

  • Usuário ter realizado um cadastro no sistema. UC001
  • Usuário ter realizado autenticação no sistema. UC002

Caminho no sistema

  • Página Inicial > Entrar > Serviço

Fluxo principal

  1. O usuário preenche o período da reserva.
  2. O usuário clica no botão 'Reservar agora'.
  3. O sistema exibe uma mensagem de sucesso.
  4. O sistema inclui a reserva na seção 'Serviços Contratados' para o cliente. UC005
  5. O sistema inclui a reserva na seção 'Serviços Reservados' para o prestador. UC005

Regras de negócio

  1. O período de reserva é obrigatório.
  2. O sistema não deverá permitir reservar o mesmo serviço no mesmo período.
